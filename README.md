# サブスク管理

サブスクリプションサービスの月額料金を登録し、毎月・年間の支出を確認できるシンプルな管理アプリです。

サービス名、月額料金、カテゴリ、更新日、メモを登録できます。登録した内容は外部サーバーには送信されず、ブラウザの `localStorage` に保存されます。

## 主な機能

- サブスクリプションの追加
- 月額料金、カテゴリ、更新日、メモの登録
- 登録済みサブスクリプションの削除
- 月額合計と年間合計の自動計算
- ブラウザ内でのデータ保存

## 使い方

アプリを開いたら、フォームにサブスクリプション情報を入力して「追加」を押します。

登録したサブスクリプションは一覧に表示され、不要になったものは「削除」から消せます。月額合計と年間合計は、登録内容に合わせて自動で更新されます。

## ローカルでの起動

依存関係をインストールします。

```bash
npm install
```

開発サーバーを起動します。

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開きます。

## コマンド

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## データの保存について

- データはブラウザごとに保存されます。
- ブラウザのサイトデータを削除すると、登録内容も削除されます。
- 認証やクラウド同期は含まれていません。
