# サブスク管理

サブスクリプションサービスの月額料金を登録し、毎月・年間の支出を確認できるシンプルなmacOSネイティブ管理アプリです。

サービス名、月額料金、カテゴリ、更新日、メモを登録できます。登録した内容はアプリ内に保存され、外部サーバーには送信されません。

## 主な機能

- サブスクリプションの追加
- 月額料金、カテゴリ、更新日、メモの登録
- 登録済みサブスクリプションの削除
- 月額合計と年間合計の自動計算
- アプリ内でのデータ保存

## 使い方

アプリを起動したら、フォームにサブスクリプション情報を入力して「追加」を押します。

登録したサブスクリプションは一覧に表示され、不要になったものは「削除」から消せます。月額合計と年間合計は、登録内容に合わせて自動で更新されます。

## ローカルでの起動

依存関係をインストールします。

```bash
npm install
```

開発版を起動します（Next.js開発サーバーとElectronアプリを同時に起動）。

```bash
npm run electron-dev
```

本番ビルド版を起動します。

```bash
npm run build
npm run electron
```

## コマンド

```bash
npm run dev          # Next.js開発サーバーのみ起動
npm run build        # Next.jsアプリをビルド
npm run start        # Next.js本番サーバー起動
npm run lint         # ESLint実行
npm run electron     # Electronアプリ起動（ビルド後）
npm run electron-dev # Electron開発版起動
```

## データの保存について

- データはアプリ内に保存されます。
- アプリをアンインストールすると、登録内容も削除されます。
- 認証やクラウド同期は含まれていません。
